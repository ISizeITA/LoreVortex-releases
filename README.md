# Loreforge — releases

This repository hosts the **auto-update artifacts** for Loreforge (signed installer + updater manifest). It is the technical backend used by the in-app updater. It is **not** the place to download Loreforge from.

## → Download Loreforge on itch.io

**[isizeita.itch.io/loreforge](https://isizeita.itch.io/loreforge)**

The itch.io page has the description, screenshots, download button, and everything you need to get started. Please use that — not this repository.

---

If you already have Loreforge installed, it will check for new versions on its own — open the start screen and click *"Check for updates"*. The app downloads the signed update in the background and reinstalls itself; nothing manual to do.
